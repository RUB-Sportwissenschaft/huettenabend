# Huettenabend — Essensumfrage

Essensumfrage fuer den Huettenabend der Schneesport-Exkursion Valmorel. Teilnehmer geben ihre Essenswahl auf dem Handy ab, Organisatoren sehen alle Antworten in einer Tabelle und koennen ein PDF exportieren.

**Live:** https://rub-sportwissenschaft.github.io/huettenabend/
**Status:** Fertig — wird jaehrlich zur Exkursion reaktiviert.

## Technische Details

| Merkmal | Wert |
|---------|------|
| Format | Single-File `index.html`, kein Build-Step |
| Deployment | GitHub Pages |
| Datenspeicherung | JSONBin.io v3 |
| Admin-Zugang | `?admin=true` (kein Passwort) |
| PDF-Export | jsPDF + AutoTable (CDN, nur im Admin) |
| Mobile | Mobile-first, ab 320px |
